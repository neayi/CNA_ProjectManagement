Ce code met à jour automatiquement les salaires mensuels dans l’onglet “Salaires collaborateurs” à partir des données importées du “Suivi mensuel des salaires”.

Pour les salaires passés :

  • Suppression de toutes les lignes “A embaucher”
  • Correction des salaires à partir de “Import Salaires”
  • Ajout des lignes manquantes à partir de “Import Salaires”

NB : pour les salaires passés, le script ne modifie que le salaire, il ne modifie pas les temps ni les statuts par exemple.

Pour les salaires futurs :

  • Suppression de tous les salaires futurs déjà enregistrés
  • Génération des salaires à partir du mois prochain, pour chacune des lignes dans “Salaires à venir”.
  • La génération prend en compte les évolutions annuelles (à la date anniversaire), les changements de statuts, etc…

NB : les lignes futures sont mises en italique
